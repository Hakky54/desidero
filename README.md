# Desidero 🔐

# Install library with:
### Install with [maven](https://mvnrepository.com/artifact/io.github.hakky54/desidero)
```xml
<dependency>
    <groupId>io.github.hakky54</groupId>
    <artifactId>desidero</artifactId>
    <version>1.0.0</version>
</dependency>
```
### Install with Gradle
```groovy
implementation 'io.github.hakky54:desidero:1.0.0'
```
